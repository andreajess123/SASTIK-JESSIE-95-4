Haii semua, selamat datang di web saya! Perkenalkan, nama saya Andrea Jessie Sutedja kelas 95 no. absen 4. Apakah kalian tau cara menggunakan media sosial dengan baik dan benar? Web ini akan menjadi sarana belajar kalian dalam penggunaan media sosial secara bijak. Web ini akan berisikan berbagai informasi terkait penggunaan media sosial, dampak positif dan negatif dari penggunaan media sosial dan cara efektif menggunakan media sosial. Apa kalian tau apa itu media sosial dan apa kegunaannya? Media sosial sendiri merupakan platform digital berbasis internet yang bertujuan untuk memfasilitasi dan membantu pengguna dalam aktivitas sehari-hari seperti berkomunikasi, bermain game, belajar (research), dan masih banyak lagi. Media sosial terdengar menarik bukan? Wajib kalian ketahui bahwa media sosial bukan hanya dapat berdampak positif pada hidup kita, namun juga dapat berdampak negatif jika tidak digunakan secara bijaksana.
